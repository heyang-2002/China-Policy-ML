# China Policy ML

Machine learning classification of Chinese State Council policy documents.

## Project Overview

This project applies machine learning methods to classify Chinese State Council policy texts into governance domains.

The dataset consists of policy documents collected from the Chinese government website.

## Methods

The project uses the following methods:

- Text preprocessing
- TF-IDF vectorization
- Supervised machine learning classification
- Model evaluation using F1 score and accuracy
